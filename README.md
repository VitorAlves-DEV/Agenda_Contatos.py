📒 Agenda de Contatos
Projeto simples de agenda de contatos desenvolvido em Python, utilizando dicionários para armazenamento e manipulação dos dados.

📋 Sobre o Projeto
A Agenda de Contatos é uma aplicação de linha de comando que permite gerenciar contatos de forma prática. O projeto foi desenvolvido como exercício prático para consolidar conceitos de estruturas de dados, funções e manipulação de dicionários em Python.

✨ Funcionalidades

➕ Adicionar contato — cadastre nome, telefone e e-mail
📋 Listar contatos — visualize todos os contatos cadastrados
✏️ Editar contato — atualize as informações de um contato existente
⭐ Marcar como favorito — destaque seus contatos mais importantes
❌ Desmarcar favorito — remova o destaque de um contato
🌟 Listar favoritos — visualize apenas os contatos marcados como favoritos


🚀 Como executar
Pré-requisitos

Python 3.x instalado — Download aqui

Rodando o projeto
bash# Clone o repositório
git clone https://github.com/VitorAlves-DEV/Agenda_Contatos.py/blob/main/agenda.py

# Acesse a pasta do projeto
cd agenda-de-contatos

# Execute o programa
python agenda.py

🖥️ Como usar
Ao executar o programa, um menu será exibido com as opções disponíveis:
Agenda de Contatos
1. Adicionar contato
2. Listar contatos cadastrados
3. Editar contato
4. Marcar contato como favorito
5. Desmarcar contato como favorito
6. Listar contatos favoritos
7. Sair
Basta digitar o número da opção desejada e seguir as instruções na tela.
